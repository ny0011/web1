```c
출처 : web1 - HTML & Internet https://opentutorials.org/course/3084

<strong> </strong> : 태그(Tag)

<h1> ~ <h6> : 제목(heading)
<br> : 줄바꿈(new line, single line break)
	원하는 만큼 줄바꿈을 주고 싶을 때 사용
<p> : 단락(paragraph)
	단락을 표현할 때 사용. CSS를 사용하면 단락 간격 조정 가능.
<img src=" "> : 이미지 삽입, src : 웹페이지 주소 혹은 로컬 이미지 파일 위치
<li> : 목록(list). 위에 ul, ol중 어느 것이 오느냐에 따라 글 앞에 적히는 문자가 달라짐
<ul> : unordered list, 순서 없음(글 앞에 점으로 표시됨)
<ol> : ordered list, 순서 있음(1. 2. 3. ...)

html 구조를 구성하는 태그
<title> : ★제목★ 중요한 태그니 꼭 쓰자
<meta> : charset 속성으로 문자 방식을 지정
<body> : 본문
<head> : 본문을 설명하는 태그
<html> : body태그와 head 태그를 감싸는 태그
<!doctype html> : 웹페이지가 html로 만들어졌다는 걸 표현
```
